<h1> API para Catálogo de Pizzas - Alunos: Kauan Cavalcante e Mateus Costa </h1>
<h2> Como executar o projeto: </h2>

<h3> 1: Clonar repositório </h3>
Realizar o clone do repositório a partir do comando:
git clone https://github.com/theuscosta7/API-Gest-o-De_Config2.git

<h3> 2: Instalar dependências </h3>
Instalando as dependências com o comando:
git install 

<h3> 3: Iniciar o servidor </h3>
Por fim, iniciar o servidor que está rodando em localhost:3000/menuPizzas:
npm start

<br>

<h1> Rotas Criadas </h1>
<h3> GET menuPizzas </h3>
Retorna a lista completa de pizzas do cardápio

<h3> POST menuPizzas </h3>
Adiciona uma nova pizza ao cardápio

<br>

<h1> Qual workflow usado? </h1>
Foi utilizado o workflow <strong> GitHub Flow </strong> por se tratar de um projeto menor contendo uma estruturação simples, onde é possível ele age de maneira mais rápida e organizada.
